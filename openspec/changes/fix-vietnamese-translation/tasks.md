# Tasks for: Fix Vietnamese Translation

1.  Create a script to scan for Thai characters in the `Language/Vi` directory.
2.  Execute the script and identify the files containing Thai text.
3.  For each identified file:
    a. Extract the Thai text.
    b. Present the Thai text to the user for translation.
    c. Receive the Vietnamese translation from the user.
    d. Replace the Thai text with the Vietnamese translation in the file.
4.  Review and confirm all translations have been applied correctly.
